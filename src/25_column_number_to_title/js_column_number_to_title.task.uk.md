Створи функцію `numberToTitle`, яка приймає додатне ціле число `num`, що представляє
номер стовпця, і повертає відповідний заголовок стовпця, як він відображається
на аркуші Excel.

Наприклад:

```
1 -> A
2 -> B
3 -> C
...
26 -> Z
27 -> AA
28 -> AB
...
```

Приклади:

```javascript
numberToTitle(1) === 'A';
numberToTitle(28) === 'AB';
numberToTitle(702) === 'ZZ';
numberToTitle(11111) === 'PKI';
```
